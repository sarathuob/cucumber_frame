Step for Running the framework, 

1. Install JDK
2. Install Eclipse
3. Download Maven and Unzil into any folder
4. Install TestNG Ad on for eclipse
5. Install Cucumber Ad on for Eclipse
6. import the project to eclipse
7. Go to Eclipse - > windows -> preferences - > add Meven path to eclipse for connecting Maven with Eclipse
8. copy driver .exe files to C:\\Driver\\
9. compile maven, and update project (force update) 
10. click on MainRUnner class and run as testNG
